Como criar um novo repositório?

1.Crie uma conta e efetue o login em sua conta GitHub conforme está escrito na página anterior.

2.Clique no botão ‘New repository’

Siga o passo a passo como mostra na imagem a seguir:

3.Digite ‘tutorial-git’ no campo ‘Repository name’

4.Selecione ‘Initialize this repository with README’ para criar um arquivo descritivo do projeto (opcional)

5.Clique no botão ‘Add .gitignore’ e selecione ‘Node’ para inicializar o projeto com a informação de arquivos e diretórios a serem ignorados no commit

6.Clique no botão ‘Create repository’
